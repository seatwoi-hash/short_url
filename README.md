# URL Shortener API

Простой сервис для сокращения URL-адресов с использованием FastAPI и SQLite.

## Возможности
Сокращение URL: Преобразование длинных URL в короткие уникальные ссылки

Перенаправление: Автоматическое перенаправление по коротким ссылкам

Статистика: Отслеживание количества переходов по каждой ссылке

База данных: Хранение данных в SQLite с автоматическим созданием таблиц

## Технологии
FastAPI - современный веб-фреймворк для создания API

SQLite3 - встроенная база данных для хранения URL

Pydantic - валидация данных

Python 3.7+ - язык программирования

## Установка
Клонируйте репозиторий или скопируйте код

Установите зависимости:

`pip install fastapi uvicorn sqlite3`

Запустите сервер:

`uvicorn main:app --reload`

docker 

`docker run -d -p 8001:80 -v shorturl_data:/app/data atatarinov12/shorturl-service:latest`
